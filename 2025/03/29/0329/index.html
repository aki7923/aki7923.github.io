學習心得

在這段時間的學習中，我接觸了許多關於人工智慧（GenAI）、Markdown、Git 以及 Python 環境設定等技術。透過探索 GenAI API 和 LLM，我對人工智慧的應用有了更深入的理解，學會了如何與 AI 模型互動，並嘗試使用 API 來開發應用程式。另外，我學習了 Markdown，這讓我能夠更有效率地撰寫文件和筆記，提升了整理資訊的能力。

在程式開發方面，我熟悉了 Git 和終端機的基本操作，這對於版本控制和開發流程至關重要。同時，透過 Python 環境的配置與實驗，我理解了軟體開發中的依賴管理與環境設定的重要性。最後，我也嘗試建立自己的部落格，這不僅加深了我對 Hexo 靜態網站的認識，也讓我能夠分享自己的學習成果。
